How it works:

  1. Enter the director's IMDB URL below
  2. Click "Add All Movies by Director"
  3. All movies by that director will be automatically added
  4. Duplicate movies already in the database will be skipped

How to find a director's IMDB URL:

  1. Go to imdb.com
  2. Search for the director's name
  3. Click on their profile page
  4. Copy the URL from your browser's address bar

The URL should look like: https://www.imdb.com/name/nm#######/
